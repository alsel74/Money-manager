# 💰 Money Manager BA

Personalni menadžer finansija za praćenje prihoda, rashoda i budžeta.

🌐 **Live demo:** https://alsel74.github.io/money-manager/

## ✨ Funkcije

- 📋 **Dnevni prikaz** – Transakcije grupirane po danima
- 📅 **Kalendar** – Månedni kalendar s iznosima po danima  
- 📊 **Mjesečno** – Godišnji pregled s tjednim razvojem
- 📋 **Sažetak** – Pregled po računima, budžet, export u Excel
- ✍️ **Opis/Bilješke** – Osobni dnevnik troškova
- 🏦 **Računi** – Gotovina, bankovni račun, kartica
- 📈 **Statistike** – Tortni grafikon po kategorijama
- 🎯 **Budžet** – Planiranje budžeta po kategorijama
- 🔒 **PIN / Face ID** – Zaštita lozinkom
- 💾 **Backup** – Export/Import JSON i CSV (Excel)
- 🗂️ **Kategorije** – Dodavanje vlastitih kategorija

## 💱 Valuta

Konvertibilna marka (KM / BAM)

## 🚀 Kako koristiti

1. Otvori https://alsel74.github.io/money-manager/ u pregledniku
2. Aplikacija radi direktno u browseru – nema instalacije
3. Podaci se čuvaju lokalno na uređaju (localStorage)

## 📱 Mobilno

Optimizirano za iPhone i Android pregledač.  
Za iOS: Safari → **Dijeli → Dodaj na početni zaslon** → radi kao nativna app!

## 🛠️ Tehnologije

- Vanilla HTML/CSS/JavaScript (bez ovisnosti)
- Progressive Web App (PWA) kompatibilno
- LocalStorage za čuvanje podataka
